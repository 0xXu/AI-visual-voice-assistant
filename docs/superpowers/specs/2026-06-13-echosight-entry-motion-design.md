# EchoSight 首屏强烈舞台动效设计

**日期**：2026-06-13

**范围**：仅修改入口首屏，不修改设备检测、实时会话、记录抽屉、结果页或业务逻辑。

## 目标

在现有深海科技视觉上强化首屏冲击力，使 EchoSight 像一个正在运行的视觉感知系统，同时保持加载轻量、交互可访问且不影响后续实时会话性能。

## 动效设计

- 标题两行采用错峰 mask reveal，强调文字增加持续但克制的渐变扫光。
- 背景增加两条低透明度空间光带、静态噪点质感和随指针移动的径向光场。
- 右侧轨道装置增加不同方向的旋转刻度、核心脉冲、扫描弧和漂浮标签。
- Header、eyebrow、说明、CTA、隐私提示和 footer 依次入场。
- CTA 在支持精细指针的设备上轻微朝指针方向偏移，并提供扫光和按压反馈。
- 指针移动通过 `requestAnimationFrame` 合并更新，只写入首屏 CSS 变量。

## 性能与降级

- 不引入 GSAP、WebGL 或新运行时依赖。
- 动画仅使用 `transform`、`opacity`、`background-position` 和 CSS 变量。
- 移动端关闭指针跟随，减少背景装饰与轨道复杂度。
- `prefers-reduced-motion: reduce` 下关闭旋转、漂移、脉冲、扫光和空间位移，仅保留静态布局。

## 验收标准

- 首屏加载后可见完整品牌、标题、说明和 CTA。
- 精细指针移动会更新首屏光场变量，离开后回到中心位置。
- CTA 点击仍只触发现有 `onStart`。
- 320px 以上视口无横向溢出。
- 其他页面组件和会话状态逻辑无改动。
